# Тестовое задание на позицию Junior AI-engineer

Финансовые аналитики тратят значительное время на изучение отчетов компаний, извлечение ключевых метрик и ответы на вопросы о финансовых показателях.

**Ваша задача** — создать прототип AI-ассистента, который помогает анализировать финансовые данные и отвечать на вопросы о результатах деятельности компании. Ассистент должен уметь читать структурированные финансовые данные и генерировать содержательные выводы.
